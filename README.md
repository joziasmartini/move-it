## 🚀 Getting started

Clone the project and access the folder

```bash
$ git clone https://github.com/joziasmartini/nlw-four.git && cd nlw-four
```

Follow the steps below
```bash
# Install the dependencies
$ yarn

# Start the project
$ yarn dev
```
